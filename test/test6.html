                                                                                                                                                                                                      Links

                                                                                                                                                                                                      To create a link, type 

                                                                                                                                                                                                      [clusterflunk link](http://clusterflunk.com/)
                                                                                                                                                                                                      and it will appear as clusterflunk link .

                                                                                                                                                                                                      Code

                                                                                                                                                                                                      To create a code block, type a blank line and then wrap your code in three tick marks(```). 

                                                                                                                                                                                                      ```
def can_haz(cheeseburger):
    if cheeseburger:
        return True
    return False
```
                                                                                                                                                                                                      def can_haz(cheeseburger):
    if cheeseburger:
        return True
    return False
